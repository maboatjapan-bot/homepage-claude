## 📋 文档维护

### 更新责任

| 文档类型 | 负责人 | 更新触发 |
|---------|--------|----------|
| 代码相关 | 开发团队 | 每次发布 |
| 内容相关 | 内容编辑 | 内容变更 |

### 文档规范

1. **Markdown 格式** - 所有文档使用 `.md` 格式
2. **版本控制** - 所有文档纳入 Git 管理
3. **变更记录** - 重要文档需要更新日期和版本号
4. **索引维护** - 新增文档时更新本文档
