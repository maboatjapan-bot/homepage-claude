## Skills とは

Skills は Claude Code AI アシスタントの拡張機能で、プログラミング知識がなくてもウェブサイトの保守作業ができます。AI がご要望を自動的に認識し、適切な手順でガイドします。

**主な特徴：**
- プログラミング知識不要
- ステップバイステップでのガイダンス
- ウェブサイトのデザイン一貫性を自動維持
- 日本語と中国語の指示に対応

## 利用可能な Skills 一覧

### 📦 add-product - 製品追加

**用途：** 新しい製品ページを作成

**使用シーン：**
- 新製品の発表（Popohu、Pitopa、Pitoshiyu など）
- 製品バリエーションの追加
- 新バージョンの製品ページ作成

**AI が以下をガイド：**
1. 製品名
2. 製品カテゴリ
3. 簡易説明
4. 詳細説明
5. 主な特徴
6. 仕様（オプション）
7. 価格情報（オプション）
8. 製品画像・動画

---

### 📰 add-news - ニュース追加

**用途：** ニュース記事やお知らせの公開

**使用シーン：**
- 新製品の発表
- 会社ニュースの更新
- イベント通知
- AI 機能のアップデート
- 採用情報

**AI が以下をガイド：**
1. 記事タイトル
2. 公開日
3. 記事カテゴリ（製品/AI/企業/その他）
4. 記事概要
5. 本文
6. アイキャッチ画像（オプション）

---

### ✏️ update-page-content - ページ内容更新

**用途：** 既存ページのテキスト内容を修正

**使用シーン：**
- 会社紹介の更新
- 製品説明の修正
- 連絡先情報の変更
- 文言の誤字修正

**AI が以下をガイド：**
1. 編集するページの選択
2. 修正箇所の特定
3. 新しい内容の入力
4. 変更の確認と適用

**対応ページ：**
- トップページ
- 会社概要
- 製品ページ
- ソリューションページ
- AI ページ
- お問い合わせ
- 採用情報

## Skills の使い方

### 方法 1：自然言語で指示（推奨）

自然な言葉でニーズを伝えると、AI が適切な Skill を自動選択します：

```
指示の例：
• 「新製品 Popohu Pro を追加して」
• 「新製品のニュースリリースを公開して」
• 「会社ページの紹介文を更新して」
• 「Popohu Mini の製品説明を修正して」
```

### 方法 2：Skill を直接指定

使用する Skill を明示的に指定します：

```
指示の例：
• 「add-product を使って製品を追加」
• 「add-news でニュースを公開」
• 「update-page-content でページを修正」
```

## よくある質問

### Q: プログラミングの知識がなくても使えますか？

**A:** もちろんです！これらの Skills は非技術者の方のために設計されています。コンテンツをご提供いただければ、AI がすべての技術作業を処理します。

### Q: 何かソフトウェアをインストールする必要がありますか？

**A:** Claude Code CLI をインストールする必要があります。公式ドキュメントを参照してください。

### Q: 変更後に取り消すことはできますか？

**A:** はい。すべての変更は Git バージョン管理システムに記録されるため、いつでも以前のバージョンに戻せます。

### Q: 変更はウェブサイトにすぐに反映されますか？

**A:** 変更の反映には以下の手順が必要です：
1. AI による修正完了
2. GitHub へのプッシュ
3. AWS Amplify による自動デプロイ
4. 数分後にウェブサイトで反映

### Q: 画像はどこに配置すればいいですか？

**A:** `public/wp-content/uploads/YYYY/MM/` ディレクトリに画像をアップロードしてください。AI が正しいパスを案内します。

### Q: 複数のページを一度に修正できますか？

**A:** はい。ページを一つずつ指定していただければ、AI が順番にすべての修正を行います。

### Q: AI が意図を勘違いした場合は？

**A:** お気軽に訂正してください。例：「違う、xxx の部分を修正したい」「xxx の效果が欲しい」

### Q: Excel や Word 文書からのインポートはできますか？

**A:** 現在は直接インポートに対応していません。内容をコピーして会話に貼り付けてください。AI がフォーマット変換を支援します。

## テクニカルサポート

問題が発生した場合：

1. **エラーメッセージを確認** - AI が詳細なエラー情報と解決策を提案します
2. **プロジェクトドキュメントを参照** - `SKILLS_MANUAL.md` で詳細な技術情報を確認
3. **開発チームへ連絡** - 解決しない場合は開発チームへご連絡ください

## ベストプラクティス

### ✅ 推奨される方法

- Skills を使用する前に、すべてのテキストと画像を準備
- 明確で具体的な説明を使用
- AI のガイダンスに従って情報を段階的に提供
- 修正完了後、ブラウザでプレビューを確認
- 定期的に変更を GitHub にプッシュ

### ❌ 避けるべき方法

- 一度に过多の情報を提供しない
- AI の検証ステップをスキップしない
- （Astro の知識がない場合）AI が作成したファイルを手動で修正しない
- 就業時間外に重大な変更をプッシュしない
