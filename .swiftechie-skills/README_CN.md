## 什么是 Skills

Skills 是 Claude Code AI 助手的扩展功能，可以帮助您完成网站维护任务，无需编写代码。AI 会自动识别您的需求，并引导您完成相应的操作。

**主要特点：**
- 无需编程知识
- 步骤化引导操作
- 自动保持网站风格一致
- 支持中文和日文指令

## 可用的 Skills 列表

### 📦 add-product - 添加新产品

**用途：** 创建新的产品页面

**使用场景：**
- 发布新产品（Popohu、Pitopa、Pitoshiyu 等）
- 添加产品变体或新版本
- 创建产品介绍页面

**AI 会引导您提供：**
1. 产品名称
2. 产品类别
3. 简短描述
4. 详细说明
5. 主要特性
6. 规格参数（可选）
7. 价格信息（可选）
8. 产品图片/视频

---

### 📰 add-news - 添加新闻

**用途：** 发布新闻文章或公告

**使用场景：**
- 发布新产品公告
- 公司新闻更新
- 活动通知
- AI 功能更新
- 招聘信息

**AI 会引导您提供：**
1. 文章标题
2. 发布日期
3. 文章分类（产品/AI/企业/其他）
4. 文章摘要
5. 正文内容
6. 配图（可选）

---

### ✏️ update-page-content - 修改页面内容

**用途：** 修改现有页面的文字内容

**使用场景：**
- 更新公司介绍
- 修改产品说明
- 调整联系信息
- 修正文案错误

**AI 会引导您：**
1. 选择要编辑的页面
2. 定位要修改的章节
3. 输入新的内容
4. 确认并应用更改

**支持的页面：**
- 首页
- 公司概要
- 产品页面
- 解决方案页面
- AI 页面
- 联系我们
- 招聘信息

## 如何使用 Skills

### 方式一：自然语言触发（推荐）

直接用自然语言描述您的需求，AI 会自动选择合适的 Skill：

```
示例指令：
• "请帮我添加一个新产品 Popohu Pro"
• "我想发布一篇新产品的新闻"
• "更新一下公司页面的介绍"
• "修改 Popohu Mini 的产品说明"
```

### 方式二：明确指定 Skill

直接告诉 AI 要使用哪个 Skill：

```
示例指令：
• "使用 add-product 添加新产品"
• "用 add-news 发布新闻"
• "用 update-page-content 修改页面"
```

## 常见问题

### Q: 我不懂编程，可以用这些 Skills 吗？

**A:** 完全可以！这些 Skills 专为非技术人员设计，您只需要提供内容，AI 会处理所有技术细节。

### Q: 需要安装什么软件吗？

**A:** 您需要安装 Claude Code CLI。请参考官方文档进行安装。

### Q: 修改后可以撤销吗？

**A:** 可以。所有更改都会被 Git 版本控制系统记录，可以随时恢复到之前的版本。

### Q: 可以直接在网站上看到更改吗？

**A:** 更改需要通过以下步骤发布：
1. AI 完成修改
2. 推送到 GitHub
3. AWS Amplify 自动部署
4. 几分钟后在网站上生效

### Q: 图片应该放在哪里？

**A:** 请将图片上传到 `public/wp-content/uploads/YYYY/MM/` 目录。AI 会提醒您正确的路径。

### Q: 可以同时修改多个页面吗？

**A:** 可以，每次指定一个页面，AI 会逐步完成所有修改。

### Q: 如果 AI 理解错了我的意图怎么办？

**A:** 您可以直接纠正 AI，例如：「不对，我是想修改xxx部分」或「我想要的是xxx效果」。

### Q: 支持 Excel 或 Word 文档导入吗？

**A:** 目前不支持直接导入。请将内容复制粘贴到对话中，AI 会帮助您转换格式。

## 最佳实践

### ✅ 推荐做法

- 在使用 Skills 前，准备好所有文字内容和图片
- 使用清晰、具体的描述
- 逐步提供信息，跟随 AI 的引导
- 修改完成后在浏览器中预览效果
- 定期推送更改到 GitHub

### ❌ 避免这样做

- 不要一次性提供过多信息
- 不要跳过 AI 的验证步骤
- 不要手动修改 AI 创建的文件（除非您了解 Astro）
- 不要在非工作时间推送重大更改
