## 🔄 部署后更新流程

### 小更新（文本修改等）

```bash
git add .
git commit -m "Update content"
git push
# → 自动触发部署
```

### 大更新（新功能、新页面）

1. 开发新功能
2. 本地测试 (`npm run build` + `npm run preview`)
3. 提交代码
4. 推送到 GitHub
5. 验证自动部署

### 紧急修复（Hotfix）

1. 修复代码
2. 快速测试
3. 提交并推送
4. 在 Amplify Console 可手动触发重新部署
